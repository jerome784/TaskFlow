# TaskFlow 📋

TaskFlow is a Team Task Management SaaS designed to give teams a central place to create, assign, track, and report on tasks with role-based access.

## Features
- User registration/login with JWT
- Create/assign tasks to team members
- Task status tracking (Todo -> In Progress -> Done)
- Priority levels
- Due dates with overdue detection
- Manager dashboard with analytics
- Reports export

## Tech Stack
- Frontend: React 18, Vite, Tailwind CSS, React Query, Zustand
- Backend: Java 17, Spring Boot 3
- Data: PostgreSQL
- DevOps: Docker, K8s, Terraform, Jenkins, Prometheus, Grafana
